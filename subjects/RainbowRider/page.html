Le chevaucheur d'arcs-en-ciel

Tobi veut se rendre chez son ami xileF, le fameux danseur, mais celui-ci n'habite pas à côté... Fort heureusement, un système d'arcs-en-ciel permet de se déplacer de ville en ville facilement. Étant donné le système d'arcs-en-ciel, indiquer en combien de sauts Tobi pourra arriver à sa destination.

Input:

Sur la première ligne, deux entiers N et D séparés par un espace, N le nombre d'arcs-en-ciel et D est le numéro de la ville dans laquelle habite xileF (1 ≤ N ≤ 10^6, 1 ≤ D ≤ 10^9).
Sur chacune des N lignes suivantes se trouvent deux entiers A et B représentant un arc-en-ciel entre les villes A et B (1 ≤ A, B ≤ 10^9, A ≠ B). Un arc-en-ciel peut se parcourir dans les deux sens.

Output

S'il y a une suite d'arcs-en-ciel entre la ville de numéro 1 et D, afficher le nombre minimal d'arcs-en-ciel que Tobi devra prendre pour se rendre chez le fameux danseur. Sinon, afficher "NOPE".

Exemples

Input

3 4
1 2
2 3
3 4

Output

3

Input

3 2
4 3
1 5
3 5

Output

NOPE
